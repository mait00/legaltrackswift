# Быстрое добавление файлов

В Xcode (уже открыт):
1. Правой кнопкой на "LegalTrack" (синяя иконка папки)
2. "Add Files to 'LegalTrack'..."
3. Перейдите в LegalTrack/LegalTrack/
4. Выберите все папки: App, Core, Features, Shared
5. ✅ Copy items if needed
6. ✅ Create groups  
7. ✅ Add to targets: LegalTrack
8. Add

Или выберите всю папку LegalTrack/ сразу.
